# RADIX Talent Match: Production-Grade Intelligent Talent Matching Platform

An enterprise-grade, single-database, unified full-stack system designed to execute talent assessments, parse resume and job descriptions (JDs) via LLM extraction, evaluate candidate benchmark levels (Talent Check), and perform semantic, exact, and fuzzy skill matching.

---

## 🚀 Key Features

1. **Intelligent Document Parsing (JD Analytics & Resume Parsing)**
   - Pipeline supporting PDF, Word (DOCX with tables), and raw TXT uploads.
   - Dual-stage extraction using **Gemini 2.5 Flash** (Primary) and **Claude 3.5 Sonnet** (Fallback) via OpenRouter.
   - SHA-256 deduplication and database-backed caching to prevent redundant API consumption and costs.

2. **Unified Profile & Skill Catalog (Profile Builder)**
   - Unified candidate schema supporting core metadata, education, and resume parsing merges.
   - Optimistic concurrency control using version numbers to prevent race conditions during updates.
   - Multi-category skill catalog tracking 12 enterprise competency codes (COD, DSA, OOD, APTI, COMM, AI, CLOUD, SQL, SWE, SYSD, NETW, OS).

3. **Deterministic Benchmarking (Talent Check)**
   - Ingests company skill benchmarks idempotently from spreadsheets.
   - Calculates candidate competency levels (1-9) dynamically using skill counts, confidence values, and activity-based evidence bonuses (internships, hackathons, certifications).
   - Category gap analyzer comparing candidate levels directly to corporate requirements.

4. **3-Tier Matching Engine (Skill Matching)**
   - Exact String Matching.
   - Fuzzy String Matching (`rapidfuzz`) with token set ratio thresholds.
   - Semantic Matching using local `all-MiniLM-L6-v2` SentenceTransformers for word vector similarity.
   - Generates a composite matching score and identifies required vs. nice-to-have missing skills.

5. **Cyber-Dark Frontend UI**
   - High-fidelity Glassmorphism design system built with React, TypeScript, Vite, and Vanilla CSS.
   - Interactive dashboards for profile settings, drag-and-drop document upload, company gap visualizations, matcher analysis, and historic runs.

## 🛠️ Installation & Setup

### Prerequisites
- Python 3.9+
- Node.js 18+
- OpenRouter API Key (placed in `.env` or system variables as `OPENROUTER_API_KEY`)

### Backend Setup
1. Navigate to the root directory.
2. Install Python dependencies:
   ```bash
   pip install -r backend/requirements.txt
   ```
   *(Ensure libraries like `fastapi`, `sqlalchemy`, `pydantic-settings`, `python-jose`, `passlib`, `rapidfuzz`, `sentence-transformers`, `openpyxl`, `pypdf`, and `python-docx` are installed)*
3. Create a `.env` file in the root or set environment variables:
   ```env
   OPENROUTER_API_KEY=your-api-key-here
   JWT_SECRET=your-jwt-secret-here
   DATABASE_URL=sqlite:///./radix.db
   ```
4. Run the company benchmark spreadsheet ingestion:
   ```bash
   python scripts/ingest_companies.py
   ```
5. Start the FastAPI server:
   ```bash
   python -m uvicorn backend.app.main:app --host 127.0.0.1 --port 8000
   ```

### Frontend Setup
1. Navigate to the `frontend/` folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Access the web application at `http://localhost:5173`.

## 🧪 Testing and Verification

### 1. Automated Regression Suite
We provide a comprehensive regression test suite that processes all 6 sample JDs and 4 resumes end-to-end. It saves a markdown table of matches and checks to `docs/regression_results.md`.
Run it from the project root:
```bash
python tests/run_regression_suite.py
```

### 2. HTTP Endpoint Verification
To verify the REST API interfaces programmatically:
```bash
python tests/test_api_endpoints.py
```
This tests user registration, logins, profile creation, and talent benchmarking runs.
